Ejercicio nro 15 Unidad 3

Se le solicita al usuario que ingrese los
extremos de un rango numérico y una cantidad
de valores desconocidos.Realice un programa para
informar si cada número está dentro del rango. Si
está en rango, informar si el número es par. Si no
está dentro del rango, informar si es número es impar.
Tenga en cuenta que el rango debe tener una diferencia
mínima de 5 números enteros.
